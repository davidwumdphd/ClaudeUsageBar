  <action>
Add two helper functions to the UsageView struct:

1. `formatPacingDelta(_ delta: Double) -> String`
   - Round to Int using `Int(delta.rounded())`
   - If rounded > 0: return "+\(rounded)% ahead of pace"
   - If rounded < 0: return "\(rounded)% behind pace" (negative sign is automatic from Int)
   - If rounded == 0: return "0%"
   - Do NOT duplicate the negative sign (no "-\(rounded)")

2. `pacingColor(_ delta: Double) -> Color`
   - Round first: `let rounded = Int(delta.rounded())`
   - If rounded > 10: return .red
   - If rounded < -10: return .green
   - Otherwise: return .secondary
