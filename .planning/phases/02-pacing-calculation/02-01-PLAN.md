   - `@Published var weeklyPacingDelta: Double = 0.0`

   - `private var pacingTimer: Timer?`

4. **`calculateDelta` method** (private):
   - Signature: `private func calculateDelta(usage: Double, resetsAt: Date?, windowDuration: TimeInterval) -> Double`
   - Guard on `resetsAt` being nil -- return 0.0 if nil
   - Compute `windowStart = resetsAt.addingTimeInterval(-windowDuration)`
   - Compute `elapsed = Date().timeIntervalSince(windowStart)`
   - Clamp elapsed to `[0, windowDuration]` using `min(max(elapsed, 0), windowDuration)`
   - Compute `expectedPercent = (clampedElapsed / windowDuration) * 100.0`
   - Return `usage - expectedPercent`
   - Note: `usage` is already 0-100 scale (from `sessionUsage` Int). Do NOT use `sessionPercentage` (that is 0.0-1.0).

5. **`recalculatePacing` method** (internal, called by timer and parseUsageData):
   - Signature: `func recalculatePacing()`
   - Sets `sessionPacingDelta = calculateDelta(usage: Double(sessionUsage), resetsAt: sessionResetsAt, windowDuration: fiveHourDuration)`
   - Sets `weeklyPacingDelta = calculateDelta(usage: Double(weeklyUsage), resetsAt: weeklyResetsAt, windowDuration: sevenDayDuration)`
